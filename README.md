# TXT_BookReader
### 使用openjdk25设计的TXT阅读器

<p>使用javac TxtReader.java执行编译
<p>使用java TxtReader运行程序
<p>使用jar cfm TxtReader.jar Manifest.txt *.class编译成jar
<p>使用java -jar TxtReader.jar运行jar程序
